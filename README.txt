This program requires Python 2.7 to be installed in order to run. Additionally, all files
must be downloaded from this GitHub page for the program to run properly.

To run the code, open a command prompt window and enter any of the commands listed
in the "commands.txt" file. It is recommended to open the file in Notepad++ or a similar 
text editing software so that each command is separated on its own line.

An example of a command that may be used to run the software is as follows:

python pacman.py -1 mediumMaze -p SearchAgent -a fn=bfs
